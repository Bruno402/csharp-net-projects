# Portafolio de proyectos en C# y .NET

Este repositorio contiene una colección de proyectos pequeños desarrollados en C# y tecnologías .NET.

## Proyectos

- [CalculadoraConsola](./CalculadoraConsola)  
  Aplicación de consola que realiza operaciones matemáticas básicas.

- [GestorNotas](./GestorNotas)  
  CRUD sencillo para gestionar notas en consola.

- [ConversorUnidades](./ConversorUnidades)  
  Conversor de temperatura, longitud y peso.

- [MiniApiRest](./ApiRestMini)  
  API REST básica en ASP.NET Core para manejar productos.
