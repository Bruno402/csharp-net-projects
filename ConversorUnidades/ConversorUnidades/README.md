# Conversor de Unidades

Aplicación de consola en C# que permite convertir:
- Temperatura (Celsius ↔ Fahrenheit)
- Longitud (Metros ↔ Pies)
- Peso (Kilogramos ↔ Libras)

## Cómo ejecutar
1. Clonar el repositorio
2. Abrir la carpeta `ConversorUnidades` en Visual Studio
3. Ejecutar con `Ctrl + F5`
